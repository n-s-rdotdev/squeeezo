# Squeeezo

Research and planning repo for a desktop-first PDF compression product.

The current direction is:

- desktop app built with `React + TypeScript + Vite + Tauri v2`
- UI layer using `shadcn/ui`
- native PDF compression via a bundled sidecar binary
- macOS-specific Finder integration via a Quick Action / Action Extension

See [docs/research.md](/Users/nrakshit/Developer/nsr/projects/squeeezo/docs/research.md) for the working research document.
